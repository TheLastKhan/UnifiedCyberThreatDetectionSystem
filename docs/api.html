Welcome to the API Documentation

Unified Cyber Threat Detection System'in REST API'sini keşfedin. Email phishing tespiti, web log analizi, ve cross-platform tehdit korelasyonu için endpoints'i görüntüleyin.

Email Detection API

Emaillerden phishing ve spam tespit edin. TF-IDF + Random Forest + LIME explanation kullanarak tahmin ve açıklamalar alın.

Web Analysis API

Web loglarındaki anormal davranışları tespit edin. Isolation Forest ve pattern matching ile saldırı desenleri bulun.

Unified Analysis API

Email ve web verilerini birlikte analiz edin. Cross-platform korelasyon ile kapsamlı tehdit analizi yapın.

Reports API

Threat analysis raporlarını yönetin. Raporları alın, PDF/CSV olarak dışa aktarın, veya geçmiş analizleri görüntüleyin.
